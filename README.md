## рџ“Њ Project Overview

**ShopEZ** is a full-stack e-commerce web application developed to provide users with a seamless online shopping experience. The application allows customers to browse products, manage their shopping cart, place orders, and securely authenticate their accounts.

The platform also includes an admin dashboard where administrators can manage products, users, and orders efficiently.

ShopEZ is built using the **MERN Stack**:
- MongoDB
- Express.js
- React.js
- Node.js

The project follows a modern client-server architecture with RESTful APIs, JWT authentication, and responsive UI design.
