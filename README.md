# 🛒 ShopEZ - E-Commerce Application

![ShopEZ Banner](https://via.placeholder.com/1200x400?text=ShopEZ+E-Commerce)

## 📌 Project Overview

**ShopEZ** is a full-stack e-commerce web application developed to provide users with a seamless online shopping experience. The application allows customers to browse products, manage their shopping cart, place orders, and securely authenticate their accounts.

The platform also includes an admin dashboard where administrators can manage products, users, and orders efficiently.

ShopEZ is built using the **MERN Stack**:
- MongoDB
- Express.js
- React.js
- Node.js

The project follows a modern client-server architecture with RESTful APIs, JWT authentication, and responsive UI design.

# 🚀 Features

## 👤 User Features

✅ User Registration and Login  
✅ Secure JWT Authentication  
✅ Browse Available Products  
✅ Search Products  
✅ View Product Details  
✅ Add Products to Cart  
✅ Update Cart Quantity  
✅ Remove Products from Cart  
✅ Place Orders  
✅ View Order History  
✅ Manage User Profile  


## 🔐 Authentication Features

- User signup and login
- Password encryption using bcrypt
- JWT-based authentication
- Protected routes
- Role-based access control


## 🛍️ Product Features

- Product listing
- Product categories
- Product images
- Product details page
- Price management
- Stock management


## 🛒 Cart Features

- Add products to cart
- Increase/decrease quantity
- Remove cart items
- Calculate total price automatically


## 👨‍💼 Admin Features

Admin dashboard provides:

✅ Add new products  
✅ Update existing products  
✅ Delete products  
✅ Manage customer orders  
✅ View registered users  
✅ Manage inventory
